# SimuLinky
That tool is a rudimentary development for simulating data consumption from Linky Wattmetter.
Build to make some tests in relation with the [Module TIC](https://github.com/bastoon577-lang/Module-TIC-SOFTWARE) project, that tool allows to simulate the behavior about SmartCharger in front of differents currents.

# How to use it ?
Run the script in Python command and inform :
 1. COM port (COM5, ttyUSB0),
 2. Type of wattmetter (Single Phase or Three phases).

Finally, you can setup up :
 1. Current by infomring value,
 2. off-peak time (HC.. or HP..)

I'm sorry for foreign people, that development was build for french wattmetter and i really don't know how data are represented in your country.
Anyway, it's very easy to change the protocole in code.
